WhatIsUp Recorder — Options

Configure the connection to your WhatIsUp instance.

Connection

The base URL of your WhatIsUp server (no trailing slash needed).

Generate a key in WhatIsUp → API Keys section. The key must start with wiu_u_.

How to get an API key
1. Log in to your WhatIsUp instance.
2. Go to Settings → API Keys.
3. Click New key, give it a name (e.g. "Recorder extension").
4. Copy the generated key and paste it above.

The key is sent as Authorization: Bearer wiu_u_… and only creates monitors on your behalf.
